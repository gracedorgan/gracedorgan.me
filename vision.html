Google Vision Accuracy Test

Personal Project

Background

I wanted to know how accurate the Google Vision image identification API actually was, according to the general public. In order to find out, I created a simple react app that should help me find out. Once I make an initial call to the Google Vision API to populate the Mongo database in this app, the app just presents picutes and their assigned tags to users. Users can decide whether or not the tag is correct. Multiple users can input their opinions, as the database is hosted on Heroku. Once all of the photos in the databse have been assessed by users, the data is displayed in a graph generated with d3.js.

My Contribution

This was just a personal project, so all code is mine.

What Went Well

I came into this project with a basic understanding of React, so setting up the interaction between the frontend and backend went smoothly and quickly.

What Could Have Gone Better

Although I understood how to pass data around in React, I still struggled with passing in a new picture every time the user clicked the next button.

What I Learned

I learned how to use the Google Vision API, and I learned how to more effectively use models and controllers in React.
