PURVEYOR OF FINE CODE

I am a computer science major at Dartmouth, with a motley assortment of interest, activities, and goals. Computer science is the root of my personality (dorky) and the source of my aspirations (lofty) since I was 10 and first realized that someone knew how to make computers and tell them what to do, even if I wasn't sure how. Sometimes I'm still not sure how. Luckily, I can, and will, learn. I like to focus on continual, fast-paced improvement of my skills without focusing too much on failure. I strive to not only be a better programmer, but also a better person. After all, why wouldn't you be both?

  • Tiny Search Engine

    Software Design and Implementation

    Used BASH scripts, Linux and Vim to write and test custom Set, Bag, and Hashtable data structures written in C to make an Index structure linking webpages and key words. The crawler, indexer and user interface components of the search engine successfully worked together to make a working search engine when used on a “closed loop” of linked pages

  • Parts of Speech Recognizer

    Data Structures and Algorithms

    Utilized a hidden Markov model, a Viterbi algorithm and the Brown corpus to build a program in Java that parsed sentences and tagged each word as a part of speech (verb, noun, etc)

  • Vibes (app)

    Web Design and Development

    A mobile app for creating location-based collaborative playlists. Built with a team of four other women. We used the spotify API, Axios, Redux, React Native, mongoDB, JWT, passport, and Babel to make it all work.

  • Google Vision Accuracy Test

    Personal Project

    A mostly frontend web app that lets users go through a dataset generated by google vision's photo tagging api and assess accuracy. I used React, Redux, Babel, Axios, and a little bit of the Google Vision API to populate my database.
